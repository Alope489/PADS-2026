# Results layout

- `results/performance/` — benchmark CSVs only (`*_data.csv`, etc.).
- `results/figures/performance/` — performance benchmark PNGs (`*_pypy.png`), produced from those CSVs by `plot_performance_no_titles.py`.
- `results/figures/multi_hop_HOL/` — multihop overlay PNG.
- `results/figures/original_fluid/` — two-flow original-fluid overlay PNG.
- `results/plots/` — replay plots where `multi-flow.png` is the three-flow run and `flow.png` is the two-flow run.
- `results/run_manifest.json` — key output paths after `./artifact`.
